About

This project was setup for study and demonstration purposes of following topics:

Angular

One of my main motivations with this website was to learn Angular web framework. This site demonstrates that learning plenty. I am using Angular Material UI framework for the basic components and look of the web page.

Accessibility

The website is designed with accessibility in mind. It can be used with a mouse, keyboard, or screen reader, such as VoiceOver for Mac. All functionalities are accessible via keyboard navigation and screen readers, ensuring usability even for blind users.

Accessibility test automation

The project is using Playwright as test automation tool with @axe-core/playwright library to automate Accessibility checks. The tests can be configured to validate specific WCAG versions, e.g. WCAG 2.1 with level AA.

Cloudinary CDN
